# 🚢 Titanic Survival Prediction (Machine Learning)

This project is a complete machine learning workflow for predicting passenger survival on the Titanic dataset using **pandas** and **scikit-learn**.

---

## 📌 Project Overview

The goal of this project is to build a binary classification model that predicts whether a passenger survived the Titanic disaster.

This project covers:
- Data loading and exploration
- Data cleaning and preprocessing
- Feature engineering
- Model training
- Model evaluation
- Kaggle submission

---

## 🧠 What I Learned

- Data preprocessing with pandas
- Handling missing values
- Feature encoding (categorical → numerical)
- Building machine learning models with scikit-learn
- Evaluating model performance
- Creating Kaggle submissions

---

## 📊 Model Performance

- **Accuracy (Kaggle Public Score): ~0.75**
